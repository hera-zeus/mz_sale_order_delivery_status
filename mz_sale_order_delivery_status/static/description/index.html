Sale order Delivery status

A Module for adding delivery status on sales order.

Basic Features
  • Available in Odoo 12.0 Community and Enterprise.
  • Allows you to see the Delivery status in Sales order.
  • Delivery Status on Sale order
    1. Draft: if the all delivery order is in Draft state.
    2. Waiting: if the all order is in Waiting state.
    3. Ready to Ship: if the all order is in Ready state.
    4. Partially Delivered: if multiple delivery order of sale order and any single delivery order is in Done State.
    5. Delivered: if the all delivery order is in Done state.
    6. Cancelled: if the all delivery order is in Cancelled state
  • No Additional Configuration is needed.
